常见问题解答

真的免费吗？

是的，完全免费。没有高级计划、没有试用期、没有隐藏费用。我们因为自己需要所以开发了它。

需要创建账户吗？

不需要。安装后立即开始使用。无需邮箱、注册或密码。数据本地存储在浏览器中。

支持哪些AI工具？

Prompting Ninja可用于任何网站。它可以直接将提示词插入ChatGPT、Claude、Gemini和标准文本框 - 包括Gmail。对于其他工具，一键复制到剪贴板在所有地方都能工作。

可以跨设备同步提示词吗？

可以，可选的。在设置中开启Chrome同步，你的提示词会跟随Chrome配置文件在所有电脑上同步。无需额外账户 - 使用Chrome内置同步。

我的数据是私密的吗？

完全私密。所有提示词本地存储在浏览器中。我们没有服务器、不收集分析、不查看你的数据。扩展只请求最少的必要权限。

可以导入现有的提示词吗？

可以。从JSON导入，或从我们的39个提示词库开始。你也可以随时导出整个库。

换浏览器怎么办？

将库导出为JSON，然后在任何基于Chrome的浏览器（Chrome、Edge、Brave、Arc）的Prompting Ninja中导入。

How to install

  1. Download the zip file and unzip it anywhere on your computer.
  2. Open Chrome and go to chrome://extensions (paste it into the address bar).
  3. Turn on Developer mode (toggle in the top-right corner).
  4. Click Load unpacked and select the unzipped folder.
  5. Done! Click the shuriken icon in your toolbar to start using Prompting Ninja.
