FAQ

Common questions

Is it really free?

Yes, completely free. No premium tier, no trial period, no hidden costs. We built this because we needed it ourselves.

Do I need to create an account?

No. Install it and start using it immediately. No email, no sign-up, no password. Your data is stored locally in your browser.

Which AI tools does it work with?

Prompting Ninja works with any website. It can insert prompts directly into ChatGPT, Claude, Gemini, and standard text fields - including Gmail (which now includes Gemini). For any other tool, one-click copy to clipboard works everywhere.

Can I sync my prompts across devices?

Yes, optionally. Turn on Chrome Sync in settings and your prompts travel with your Chrome profile across all your computers. No account needed - it uses Chrome's built-in sync.

Is my data private?

Completely. All prompts are stored locally in your browser. We don't run servers, collect analytics, or see your data. The extension requests only the minimum permissions needed to function.

Can I import my existing prompts?

Yes. Import from JSON, or start with our curated library of 39 prompts and build from there. You can also export your entire library at any time.

What if I switch browsers?

Export your library as JSON and import it into Prompting Ninja on any other Chrome-based browser (Chrome, Edge, Brave, Arc).
