Privacy Policy

Last updated: 2 April 2026

Summary

Prompt Vault does not collect, transmit, or share any personal data. All data is stored locally in your browser.

Data storage

All prompts, settings, and preferences are stored locally on your device using Chrome's built-in storage (chrome.storage.local). If you enable the optional "Sync across devices" feature, your data is synced via Google's Chrome Sync infrastructure — Prompt Vault does not operate any servers.

Data collection

Prompt Vault does not collect:

  • Personal information
  • Browsing history
  • Analytics or usage data
  • Crash reports

Permissions

Prompt Vault requests only the minimum permissions needed to function:

  • storage — to save your prompts and settings locally
  • activeTab — to insert prompts into the current page when you choose to
  • clipboardWrite — to copy prompts to your clipboard
  • scripting — to inject prompts into supported web pages

Third-party services

Prompt Vault does not use any third-party services, APIs, or analytics platforms.

Changes to this policy

If we make changes to this privacy policy, we will update this page and the "Last updated" date above.

Contact

If you have questions about this privacy policy, please open an issue at github.com/krystianzajac/prompt-vault/issues.
