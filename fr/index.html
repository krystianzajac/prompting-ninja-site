FAQ

Questions frequentes

C'est vraiment gratuit?

Oui, completement gratuit. Pas de plan premium, pas de periode d'essai, pas de couts caches. Nous l'avons construit parce que nous en avions besoin nous-memes.

Dois-je creer un compte?

Non. Installez et commencez a utiliser immediatement. Pas d'email, pas d'inscription, pas de mot de passe. Vos donnees sont stockees localement dans votre navigateur.

Avec quels outils IA ca fonctionne?

Prompting Ninja fonctionne avec n'importe quel site web. Il peut inserer des prompts directement dans ChatGPT, Claude, Gemini et les champs texte standard - y compris Gmail. Pour tout autre outil, la copie en un clic fonctionne partout.

Puis-je synchroniser mes prompts entre appareils?

Oui, optionnellement. Activez Chrome Sync dans les parametres et vos prompts suivent votre profil Chrome sur tous vos ordinateurs. Sans compte supplementaire - utilise la synchronisation integree de Chrome.

Mes donnees sont-elles privees?

Completement. Tous les prompts sont stockes localement dans votre navigateur. Nous n'avons pas de serveurs, ne collectons pas d'analytiques et ne voyons pas vos donnees.

Puis-je importer mes prompts existants?

Oui. Importez depuis JSON, ou commencez avec notre bibliotheque de 39 prompts. Vous pouvez aussi exporter toute votre bibliotheque a tout moment.

Et si je change de navigateur?

Exportez votre bibliotheque en JSON et importez-la dans Prompting Ninja sur n'importe quel navigateur base sur Chrome (Chrome, Edge, Brave, Arc).

How to install

  1. Download the zip file and unzip it anywhere on your computer.
  2. Open Chrome and go to chrome://extensions (paste it into your address bar).
  3. Turn on Developer mode using the toggle in the top-right corner.
  4. Click Load unpacked and select the unzipped folder.
  5. Done! Click the shuriken icon in your toolbar to start using Prompting Ninja.
